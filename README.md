# Real-Time Stock Portfolio Manager

A backend system for managing stock portfolios with secure authentication, portfolio tracking, and real-time stock price updates.

## Tech Stack

- Spring Boot
- PostgreSQL
- Spring Security
- JWT Authentication
- Spring Data JPA (Hibernate)
- Docker
- React (planned)

## Features

- User Authentication (JWT)
- Portfolio Management
- Buy/Sell Transactions
- Real-Time Stock Price Tracking
- Scheduled Price Updates
- Pagination & Filtering
